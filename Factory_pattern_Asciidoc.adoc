## Yarartýmsal(Creational) tasarým desenleri:
Bu paternler, yazýlým nesnelerinin nasýl yaratýlacaðý hakkýnda genel olarak öneriler sunarak kullandýðý esnek yapý sayesinde daha önceden belirlenen durumlara baðlý olarak gerekli nesneleri yaratýr. Yaratýmsal(Creational) desenler, hangi nesnenin çaðrýlmasý gerektiðini izlemeden sistemin uygun nesneyi çaðýrmasýný saðlayan tasarým kalýplarýdýr. Nesnelerin yaratýlmasý gerektiði durumlarda uygulamaya farkedilebilir bir esneklik katar. Esas amaç, iyi bir yazýlýmýn içinde barýndýrdýðý nesnelerin nasýl yaratýldýðýndan baðýmsýz olarak tasarlanmasý gerekliliðidir.

### * Factory Pattern: 
Bazý nesneler doðrudan kullanýcý tarafýndan yaratýlmasý istenmemektedir(new ile yartma iþlemi). Yani bazý nesneler kullanýcýdan gizlenmiþtir. 
Asýl amaç, oluþturmak istediðimiz sýnýfýn kendisinden bir örnek istemek yerine Factory Metod patterni sayesinde tek bir instance üzerinden gerekli nesnenin üretilmesini saðlamaktýr. Algoritmada belirli adýmlarda farklý tiplerde nesnelere ihtiyacýnýz olabilir. Bu nesneleri if – else veyahut switch yapýlarý aracýlýðýyla, duruma göre oluþturabiliriz. Örneðin SystemElementFactory sýnýfýnýn createKullanici() isimli üyesi factory method durumundadýr. Kullanici sample = SystemElementFactory.createKullanici(); kullanici türünden nesne geri döndürülür.

package FactoryPattern_Example_2;

public abstract class Kullanici {
	
	private String kullanici_turu;
	private int age;
	
	public void setKullanici_turu(String kullanici_turu){
		this.kullanici_turu = kullanici_turu;

	public void setAge(int age){
		this.age = age;
	}
	public String getKullanici_turu(){
		return this.kullanici_turu;
	}
	public int getAge(){
		return this.age;	
	}
	
	
	public void showUp(){
		System.out.printf("Kullanýcý türü :[%s], Kullanýcý Yaþý:[%d]\n", getKullanici_turu(),getAge());
	}
	
}

public class SystemElementFactory {

	static Random random = new Random();
	
	public static void main(String[] args) {
		System.out.println("### Factory Design Pattern's Example is Runing ###\n");

		// Random olarak olusturulacak
		int loop=7;
		while(loop>0){
			Kullanici kullanici = SystemElementFactory.createKullanici(getRandom(1,2));
			kullanici.showUp();
			loop--;
		}

	}
	public static int getRandom(int min, int max){
		return random.nextInt((max-min)+1)+ min;
	}

}
