## Yarartýmsal(Creational) tasarým desenleri:
Bu paternler, yazýlým nesnelerinin nasýl yaratýlacaðý hakkýnda genel olarak öneriler sunarak kullandýðý 
esnek yapý sayesinde daha önceden belirlenen durumlara baðlý olarak gerekli nesneleri yaratýr. Yaratýmsal(Creational) desenler, 
hangi nesnenin çaðrýlmasý gerektiðini izlemeden sistemin uygun nesneyi çaðýrmasýný saðlayan tasarým kalýplarýdýr. 
Nesnelerin yaratýlmasý gerektiði durumlarda uygulamaya farkedilebilir bir esneklik katar. Esas amaç, 
iyi bir yazýlýmýn içinde barýndýrdýðý nesnelerin nasýl yaratýldýðýndan baðýmsýz olarak tasarlanmasý gerekliliðidir.

### * Abstract Factory Pattern: 
Benzer özelliklere sahip objeler varsa, veya birden fazla ürün ailesi çalýþmak zorunda kaldýðýmýzda kullanacaðýmýz ürün ailesi ile istemci tarafý soyutlamak için kullanýlýr
"Abstract Factory" birbirleriyle iliþkili yada birbirlerine baðlý olan nesnelerin oluþturulmasýný en etkin bir þekilde çözmeyi hedefler. 
Bu hedefe ulaþmak için soyut sýnýflardan(abstract class) veya arayüzlerden(interface) yoðun bir þekilde faydalanmaktadýr. 
"Abstract Factory" deseninin ana temasý belirli sýnýflarýn içerdiði ortak arayüzü soyut bir sýnýf yada arayüz olarak tasarlamaktýr. 
Böylece nesneleri üreten sýnýf, hangi nesnenin üretileceði ile pek fazla ilgilinmesi gerekmez. Ýlgilenmesi gereken nokta oluþturacaðý 
nesnenin hangi arayüzleri desteklediði yada uyguladýðýdýr. Bahsi geçen mekanizmalarla deseni oluþturduðumuz anda çalýþma zamanýnda hangi 
nesnenin oluþturulmasý gerektiðini bilmeden nesnelerin oluþturulmasýný yönetebiliriz. 

Eðer bir nesne oluþturacaksanýz ve tam olarak hangi nesnenin oluþturulacaðýna bir switch yada if deyimi ile karar veriyorsanýz muhtemelen 
her nesneyi oluþtruduðunuzda ayný switch yapýsýný kullanmak zorunda kalacaksýnýz. Bu tür tekrarlarý önlemek için "Abstarct Factory" deseninden 
faydalanýlabilir. Bu elbetteki nesnelerin ortak bir arayüzü uygulamýþ olma zorunluluðunun getirdiði bir faydadýr.

package AbstractFactory_Pattern;

public class FactoryProducer {
	
	final static String KULLANICI = "Kullanici";
	final static String PARASALISLEM = "Parasalislemi";
	
	public static AbstractFactory getFactory(String factory){
		if(KULLANICI.equalsIgnoreCase(factory)){
			return new KullaniciFactory();
		}else if(PARASALISLEM.equalsIgnoreCase(factory)){
			return new ParasalislemiFactory();
		}
		return null;
	}
}
